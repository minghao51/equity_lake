# Documentation History Archive

This directory stores documentation that was moved out of the active `docs/`
tree during the 2026 consolidation.

## Sections

- `backtesting/` contains archived backtesting summaries and detailed test
  writeups from the first framework rollout.
- `implementations/` contains detailed implementation writeups and feature
  summaries that are no longer front-door docs.
- `decisions/` contains historical decision notes that may still be useful for
  background.
- `planning/` contains earlier project plans and development logs.
- `research/` contains research, concept notes, and specialized guides that are
  no longer part of the main documentation flow.
- `superseded-copies/` contains older duplicate or replaced copies of docs that
  now have canonical versions under `docs/`.
